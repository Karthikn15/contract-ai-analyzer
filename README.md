# AI Contract Analysis System

## Description
AI-powered system to analyze legal contracts, detect risks, and generate reports for Indian SMEs.

## Features
- Contract Upload (PDF/DOC/TXT)
- Risk Assessment
- Clause Analysis
- Compliance Check
- PDF Report Generation

## Tech Stack
- FastAPI
- Python
- spaCy
- Transformers
- PostgreSQL (optional)

## How to Run

```bash
pip install -r requirements.txt
uvicorn contract_ai_app:app --reload
